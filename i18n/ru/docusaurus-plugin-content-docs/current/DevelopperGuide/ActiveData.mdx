# Property ActiveAccount RU

export const Highlight = ({ children, color, data }) => (
  <span
    style={{
      backgroundColor: color,
      borderRadius: "20px",
      color: "#fff",
      padding: "10px",
      cursor: "pointer",
      margin: "16px",
    }}
    onClick={() => {
      if (window.SSS === undefined) alert("SSS is not active");
      if (data === "ADDR") alert(`Your Address is ${window.SSS.activeAddress}`);
      if (data === "PUB")
        alert(`Your Public Key is ${window.SSS.activePublicKey}`);
      if (data === "NET")
        alert(`Your Network is ${window.SSS.activeNetworkType}`);
      if (data === "NAME") alert(`Your Name is ${window.SSS.activeName}`);
    }}
  >
    {children}
  </span>
);

Свойства, предоставляющие информацию о данных ActiveAccount, предоставленных SSS.

:::danger
初回画面描画時にこれらのデータを取得しようとすると、SSS オブジェクトの挿入より先に取得が行われ、値が取得できない場合があります。setTimeout 等で遅延を入れてデータを取得してください。
:::

## ActiveAddress

ActiveAccount Свойство, предоставляющее адрес учетной записи, для которой установлено значение

тип: строка

```js
// window.SSS
const address = window.SSS.activeAddress;
```

```js
// sss-module
import { getActiveAddress } from "sss-module";
const address = getActiveAddress();
```
