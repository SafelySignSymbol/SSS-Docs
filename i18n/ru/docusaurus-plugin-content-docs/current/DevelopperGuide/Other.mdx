# Другие Методы

Методы, не включенные в объекты SSS

## isAllowedSSS

- Никаких аргументов
- логическое значение возвращаемого значения

SSS が連携されているかを返却します。

## requestSSS

- Никаких аргументов
- логическое значение возвращаемого значения

Проверьте, подключен ли SSS, и если он не связан, в правом нижнем углу экрана отобразится сообщение с запросом на подключение.
