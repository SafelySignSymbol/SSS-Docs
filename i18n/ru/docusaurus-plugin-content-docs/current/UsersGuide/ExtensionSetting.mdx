# Настройка расширения

Нажмите значок в правом верхнем углу экрана и нажмите «Настройки», чтобы перейти на страницу настроек.
![メニュー](./images/menu.png)

В этом разделе описывается основные настройки для SSS Extension.

![設定](./images/setting_01.png)

## Скачать историю подписей

Загрузите полезную нагрузку подписания, выполненного с помощью расширения SSS, в файл. Можно восстановить из этой полезной нагрузки и выполнить повторное объявление.

## Сетевые настройки

Настройте сеть для использования с расширением SSS.

## Языковые настройки

Измените язык отображения текста в SSS Extension. Данный виджет в настоящее время поддерживает следующие языки:

- 日本語
- English
- 한국어
- Русский
- Italian

## Время блокировки

Вы можете установить, как долго вы будете оставаться в системе SSS Extension. Исходная настройка: 0 мин.

:::danger

Поскольку пароль хранится в течение определенного периода времени, пожалуйста, поймите, что существует риск утечки ключа.

:::

## Сброс настроек 

Вы можете инициализировать все настройки информации об аккаунте.
