# Property ActiveAccount EN

export const Highlight = ({ children, color, data }) => (
  <span
    style={{
      backgroundColor: color,
      borderRadius: "20px",
      color: "#fff",
      padding: "10px",
      cursor: "pointer",
      margin: "16px",
    }}
    onClick={() => {
      if (window.SSS === undefined) alert("SSS is not active");
      if (data === "ADDR") alert(`Your Address is ${window.SSS.activeAddress}`);
      if (data === "PUB")
        alert(`Your Public Key is ${window.SSS.activePublicKey}`);
      if (data === "NET")
        alert(`Your Network is ${window.SSS.activeNetworkType}`);
      if (data === "NAME") alert(`Your Name is ${window.SSS.activeName}`);
    }}
  >
    {children}
  </span>
);

Properties that provide information about ActiveAccount data provided by SSS.

:::danger
初回画面描画時にこれらのデータを取得しようとすると、SSS オブジェクトの挿入より先に取得が行われ、値が取得できない場合があります。setTimeout 等で遅延を入れてデータを取得してください。
:::

## ActiveAddress

ActiveAccount に設定されているアカウントのアドレスを提供するプロパティ

type: line

```js
// window.SSS
const address = window.SSS.activeAddress;
```

```js
// sss-module
import { getActiveAddress } from "sss-module";
const address = getActiveAddress();
```

<Highlight color="#25c2a0" data="ADDR">

</Highlight>

## ActivePublicKey

A property that provides the account's public key for an ActiveAccount

type: line

```js
// window.SSS
const pubkey = window.SSS.activePublicKey;
```

```js
// sss-module
import { getActivePublicKey } from "sss-module";
const pubkey = getActivePublicKey();
```

<Highlight color="#25c2a0" data="PUB">
  Show ActiveAccount PublicKey
</Highlight>

type: number

```js
// window.SSS
const networkType = window.SSS.activeNetworkType;
```

```js

import { getActiveNetworkType } from "sss-module";

```
